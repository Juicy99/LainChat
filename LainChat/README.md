#  Lain Chat

アニメserial experiments lainに登場するコンピューターネットワークWiredでの通信をイメージしたチャットアプリです。
このアプリでは全員共通のチャットに参加できます。
serial experiments lainの世界観を再現したチャットアプリを用いてserial experiments lainが好きな人同士で交流できます。
